Definición

La recursividad es una técnica de programación en la que una función se llama a sí misma directa o indirectamente para resolver un problema, descomponiéndolo en subproblemas más simples [1][2].

Procedimientos y Aplicaciones

Se utiliza cuando la solución de un problema puede representarse en términos de una versión más pequeña del mismo problema. Ejemplos comunes incluyen el cálculo de factoriales, la sucesión de Fibonacci, algoritmos de búsqueda y ordenamiento como MergeSort y QuickSort, y estructuras como árboles binarios [3][4].

Características

  • Divide un problema complejo en subproblemas más simples [5].
  • Debe tener una condición de parada o caso base [1].
  • Facilita la comprensión y claridad de ciertos algoritmos [2].
  • Puede implicar mayor consumo de memoria debido a la pila de llamadas [6].

Ejemplos de Recursividad

Factorial


function factorial(n) {
  if (n === 0) return 1;
  return n * factorial(n - 1);
}

Ejemplo: factorial(5) = 120

Fibonacci


function fibonacci(n) {
  if (n <= 1) return n;
  return fibonacci(n - 1) + fibonacci(n - 2);
}

Ejemplo: fibonacci(6) = 8

Torres de Hanói


function hanoi(n, origen, destino, auxiliar) {
  if (n === 1) {
    console.log(`Mover disco 1 de ${origen} a ${destino}`);
    return;
  }
  hanoi(n - 1, origen, auxiliar, destino);
  console.log(`Mover disco ${n} de ${origen} a ${destino}`);
  hanoi(n - 1, auxiliar, destino, origen);
}

Fractales y Recursividad

Los fractales como el triángulo de Sierpinski son un ejemplo visual de recursividad, ya que se generan repitiendo un patrón a menor escala infinitamente [3].

Conclusión

La recursividad es una técnica poderosa en ciencias de la computación, que permite soluciones elegantes y eficientes a problemas complejos. Sin embargo, debe usarse con cuidado para evitar problemas de rendimiento y desbordamiento de pila [6].
